# 英语听力测试系统

这是一个简单的在线英语听力测试系统，可以用于测试求职者的英语听力水平。

## 功能特点

- 支持5个音频文件的播放
- 自动计算单词拼写正确率
- 正确率达到80%以上显示测试通过
- 显示正确答案供参考
- 响应式设计，适配不同设备

## 使用说明

### 1. 添加音频文件

将5个音频文件命名为 `audio1.mp3`、`audio2.mp3`、`audio3.mp3`、`audio4.mp3` 和 `audio5.mp3`，并放置在项目根目录下。

### 2. 修改正确答案

在 `script.js` 文件中，找到 `correctAnswers` 数组，修改为对应的正确答案：

```javascript
const correctAnswers = [
    "第一个音频的正确答案",
    "第二个音频的正确答案",
    "第三个音频的正确答案",
    "第四个音频的正确答案",
    "第五个音频的正确答案"
];
```

### 3. 启动测试系统

1. 在项目根目录下启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```

2. 在浏览器中访问 `http://localhost:8000`

### 4. 进行测试

1. 点击音频播放器播放音频
2. 在文本框中输入听到的内容
3. 完成所有5个音频测试后，点击"提交测试"按钮
4. 查看测试结果和正确答案

## 测试结果说明

- 正确率达到80%及以上：显示"恭喜！你通过了测试！"
- 正确率低于80%：显示"很遗憾，你没有通过测试。"

## 技术实现

- **前端**：HTML5、CSS3、JavaScript
- **音频播放**：HTML5 Audio API
- **测试逻辑**：JavaScript计算单词匹配率

## 项目结构

```

```

## 自定义修改

### 修改测试题目数量

1. 在 `index.html` 中添加或删除音频项
2. 在 `script.js` 中修改 `correctAnswers` 数组长度
3. 在 `script.js` 中修改获取用户输入的逻辑

### 修改样式

编辑 `style.css` 文件，可以自定义页面颜色、字体、布局等。

### 修改通过标准

在 `script.js` 文件中，修改 `displayResult` 函数中的通过率阈值：

```javascript
if (accuracy >= 80) {  // 修改80为其他数值
    // 显示通过信息
} else {
    // 显示未通过信息
}
```

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）。

## 注意事项

1. 确保音频文件格式为MP3，以保证在大多数浏览器中正常播放
2. 音频文件大小不宜过大，建议控制在1-5MB之间
3. 正确答案应与音频内容完全匹配，包括大小写和标点符号（系统会自动转换为小写进行比较）
